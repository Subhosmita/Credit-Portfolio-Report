# Credit-Portfolio-Report
This repository contains an end-to-end, enterprise-grade Business Intelligence solution that tracks a $28.14M corporate lending portfolio. The solution features a modern cloud data stack that migrates decentralized financial records into a structured cloud data warehouse, models them into an optimized star schema, and delivers dynamic intelligence across two core reporting pages tailored for C-suite Executives and Risk Compliance Auditors.
